# 💍 Engagement Invitation Website

Traditional Indian engagement invitation  
Telugu + English  
Mobile-friendly  
Hosted on GitHub Pages

## How to use
1. Fork this repository
2. Update names, date, venue in `index.html`
3. Replace `assets/ganesha.png` and `assets/music.mp3`
4. Add your Google Form RSVP link
5. Enable GitHub Pages (Settings → Pages)
